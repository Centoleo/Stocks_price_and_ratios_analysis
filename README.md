# Stocks_price_and_ratios_analysis

This script plots financial ratios for a company and its peers for analysis purposes.
Data are downloaded using yfinance and visualized with matplotlib.
The script plots the following ratios: percentage returns of stocks, price analysis, trailing P/E, total asset turnover and revenue percentage change, net profit margin, EBITDA margin, return on equity and financial leverage, Free Cash Flow per share. 
ROE is calculated using a DuPont-style decomposition.
Pandas is used to process the data.
Some tickers might have missing or inaccurate data based on Yahoo Finance information.
